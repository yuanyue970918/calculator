# calculator
一、项目名及简介
Calculator：
这是一个可以实现整数的加、减、乘、除、开方、平方的一个10进制计算器。

二、功能
“+” ： 对10进制数做加法运算
“-”  ： 对10进制数做减法运算
“*” ： 对10进制数做乘法运算
“/” ： 对10进制数做除法运算
“AC” ： 清除一位数
“CE” ： 全出清除
“Sqrt” ：对10进制数做开方运算
“Rec ” ：对10进制数做求导运算
注：并且这个计算器能实现累加累减的连续运算。
